# RovoDev for PMMs

> *Mmes Adrienn, Daniella, Himja, Hosana, and Monica, and Messrs Andrew and Sid of the **PMM AI School** team bring to you the magic of Rovo Dev TUI — without the mess of reading the docs.*

## Quick Install

Open Terminal and paste:

```bash
bash <(curl -fsSL sidc.ai/rovodev)
```

That's it. Type `alohomora` when prompted, and the installer handles the rest.

## What it does

- Installs the Atlassian CLI (no admin rights needed)
- Authenticates you with hello.atlassian.net
- Sets up a sandboxed AI workspace with git safety net
- Installs PMM skills (getting-started, pdf-research, presentation-builder)
- Drops a Rovo Dev icon on your Desktop and Dock
- Launches the TUI

## After install

Click the **Rovo Dev** icon in your Dock or on your Desktop. No Terminal needed after the first run.

## Uninstall

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/siddharthachaturvedi/rovodev/main/rovodev-uninstall.command)
```

Or if you have the file locally:

```bash
bash rovodev-uninstall.command
```
